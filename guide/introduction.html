# Validan

Validan (Esperanto word means "valid" in english, pronounced "validan"open in new window) is a simple, performant, and powerful validation library for Node.js to validate data types and schemas, built using zero dependencies, and with a focus on simplicity.

# Features

  • Zero dependencies used ❤‍🔥.
  • Very simple to use ❤‍🔥.
  • Very performant ❤‍🔥.
  • Flexible API ❤‍🔥.
  • Easy to extend ❤‍🔥.
  • Typescript support ❤‍🔥.
